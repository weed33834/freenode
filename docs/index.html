{% include section-title.html title="快速开始" meta="从订阅到连接 · 8 步" %}

选择订阅

从上方选 Clash / V2Ray / 代理列表其中一种格式,复制链接。

安装客户端

根据操作系统在 指南页选一个客户端。

导入订阅

在客户端「订阅 / Profile」设置里粘贴订阅链接。

更新订阅

客户端会自动按间隔更新,也可手动触发。

选择节点

从节点列表里选一个延迟较低的节点连接。

开启代理

客户端通常自带系统代理开关,打开即可。

定期更新

节点过期很快,建议客户端每隔几小时自动更新订阅。

订阅失效

等下一次数据同步后恢复,或访问仓库提 Issue 反馈。
