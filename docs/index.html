{% include section-title.html title="Quick Start" meta="from subscription to connection · 8 steps" %}

Pick a subscription

Choose Clash, V2Ray or proxy list format above and copy the link.

Install a client

Pick a client on the guide page for your OS.

Import the subscription

Paste the subscription URL in the client's Profile / Subscription settings.

Update the subscription

The client auto-updates on a schedule, or you can trigger it manually.

Choose a node

Pick a low-latency node from the list and connect.

Enable proxy

Most clients have a system proxy toggle — turn it on.

Refresh regularly

Nodes expire fast; set the client to auto-update every few hours.

Subscription broken

Wait for the next sync, or open an Issue on the repo.
