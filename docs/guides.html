{% include section-title.html title="使用步骤" meta="从订阅到连接 · 8 步" %}

选择订阅链接

从首页选 Clash / V2Ray / 代理列表其中一种格式。

安装客户端

根据你的操作系统选择上方推荐客户端之一。

添加订阅

在客户端「订阅」或「Profile」设置里粘贴订阅链接。

更新订阅

客户端会自动按间隔更新,也可手动触发。

选择节点

从节点列表里选一个延迟较低的节点连接。

配置代理

客户端通常自带系统代理开关,打开即可。

定期更新

节点过期很快,建议客户端每隔几小时自动更新订阅。

故障排查

如订阅失效,等下一次数据同步后恢复,或到仓库提 Issue 反馈。
