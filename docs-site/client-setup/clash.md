# Clash 配置

## Clash Verge / Clash for Windows

1. 下载适合你操作系统的客户端。
2. 打开 **配置文件**（Profiles / 配置 / 订阅）。
3. 粘贴订阅 URL：
   ```
   https://raw.githubusercontent.com/MS33834/freenode/main/nodes/clash.yaml
   ```
4. 点击下载。
5. 进入 **代理** 选择一个节点。
6. 打开 **系统代理**。

## 故障排查

- 如果没有节点可用，等待下一次每日更新或在 Issues 中报告数据源失效。
- 确保你的客户端支持配置中的协议。
