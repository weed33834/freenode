# Shadowrocket 配置教程（iOS）

Shadowrocket 是 iOS / iPadOS 上功能强大的代理客户端，支持 V2Ray、Clash、Trojan、Shadowsocks 等多种协议，适合导入 FreeNode 的 V2Ray / Base64 订阅。

## 准备工作

1. 在 App Store 购买并下载 Shadowrocket（付费应用，无官方免费版）。
2. 确保你的 Apple ID 所在地区可以访问该应用。
3. 安装完成后打开应用，允许添加 VPN 配置。

## 配置步骤

1. 打开 Shadowrocket，点击首页右上角 **+**。
2. 类型选择 **Subscribe**（订阅）。
3. 在 URL 栏粘贴 FreeNode 的 V2Ray 订阅地址：
   ```
   {{github_url}}/nodes/v2ray.txt
   ```
4. 备注填写 `FreeNode`，点击右上角 **完成**。
5. 向左滑动刚添加的订阅卡片，点击 **更新**，等待节点下载完成。
6. 更新完成后，点击返回首页，在节点列表中选择一个节点。
7. 点击顶部全局路由开关旁的状态，选择连接模式：
   - **配置**：按规则分流（推荐日常使用）
   - **代理**：全部流量走代理
   - **直连**：全部直连
8. 点击首页顶部连接开关，允许建立 VPN 连接。

## 延迟测试

1. 在首页节点列表中，长按或点击右上角菜单选择 **延迟测试**。
2. 绿色节点表示可用，红色或超时节点建议更换。
3. 可开启 **自动测试** 与 **按延迟排序**，快速找到最优节点。

## 常见错误排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 订阅更新失败 | GitHub Raw 被 DNS 污染 | 切换到 4G/5G 或不同 Wi-Fi 重试 |
| 导入后没有节点 | 订阅格式不兼容 | 确认使用 V2Ray / Base64 格式，而非 Clash YAML |
| 所有节点超时 | 节点已失效或本地网络限制 | 等待次日更新，或尝试其他数据源 |
| 无法打开部分网站 | 规则排除该域名 | 临时切换到全局模式测试 |
| 连接开关无法打开 | VPN 配置权限被拒绝 | 删除原有 VPN 配置，重新添加并允许 |

## 安全提示

> ⚠️ 公开节点由第三方维护，运营者可能查看、记录或篡改你的流量。请仅用于学习网络协议与隐私研究，不要在免费代理/节点环境下登录银行、支付、社交等敏感账户。

- 建议启用 Shadowrocket 的 **按规则连接**，让国内网站直连，减少流量暴露。
- 敏感操作请使用可信商业 VPN 或自建节点。
- 不要共享你的订阅链接到公开渠道，避免被滥用或封禁。
