GitHub GitCode 文档

把散落各处的公开节点，
收进一条订阅链接。

FreeNode 自动抓取公开渠道的代理节点，解析、去重、校验可用性， 再统一输出成 Clash、V2Ray 或纯文本订阅。你想用哪台机器、哪个客户端， 都能直接拉一份走，不用再手动翻链接。

查看仓库 看文档

仅用于学习网络协议与隐私研究。节点来自公开来源，不保证可用性和安全性。

三种订阅格式

Clash YAML、V2Ray Base64、纯文本代理列表，主流客户端都能直接订阅。

每天自动跑一遍

定时任务抓取、解析、连通性校验、入库一条龙，过期节点自动软删。

来源完全公开

数据源列表写在 config/sources.json，谁都能看、能提 PR 补充新源。

带后端和数据库

FastAPI + SQLite 落库，前端有节点浏览、搜索、详情和管理后台。

密钥加密存储

节点认证信息 AES-GCM 加密入库，没配 key 才降级明文（仅开发用）。

一键自部署

docker compose up 起 web + 后端 + Caddy，自动 HTTPS，照着文档抄就行。

它怎么跑的

  1. 抓 — 从配置的数据源拉原始文本，支持 GitHub Raw、公开 API。
  2. 解 — 解析 vmess / vless / ss / trojan 链接，去重，丢弃私网地址。
  3. 验 — TCP 连通性 + 延迟测，可选 geo 归属地，标记存活。
  4. 存 — 按指纹去重入库，写检查日志，刷新缓存。
  5. 发 — 生成订阅文件，前端 API 也能按协议/地区过滤实时生成。
